# 텍스트 관련 태그들
 - 문단 : block레벨 태그(블록 레벨 태그)
 - 문장 : inline레벨 태그(인라인 레벨 태그)


# 텍스트를 덩어리로 묶어주는 태그
 - block레벨 태그
 - 블록 레벨 태그
 - 단락을 표시(줄 바꿈)
 - 문단이라고 생각
 - 너비와 높이가 지정 가능함. 공간이 정해져 있음
 - 외부 여백을 모든 방향에서 설정 가능. 상하좌우


# 제목을 표시하는 <hn> 태그 
 - 일반 텍스트 보다 굵고 진하게 표시됨
 - <hn> 태그는 제목을 뜻하는 heading의 줄임말인 'h'와 제목 크기를 나타내는 숫자(1~6)를 사용해서 <h1>에서 <h6> 까지 표현
 - <h1>이 가장 큰 제목, <h6>이 가장 작은 크기의 제목
 - 공간을 차지하고 너비와 높이 지정이 가능


# 단락을 만드는 <p> 태그
 - <p>태그는 paragraph의 줄임말
 - 텍스트 단락을 만드는데 이때 '단락'이란 앞뒤에 줄바꿈이 있는 텍스트 덩어리를 말함
 - 상하에 여백이 생김
 - 너비와 높이 지정 가능


# 줄을 바꾸는 <br> 태그
 - 줄을 바꿀 위치에 <br> 태그를 사용
 - <br> 태그는 break의 줄임말이고 닫는 태그가 없음


 # 수평 줄을 넣는 <hr> 태그
 - horizontal의 줄임말
 - 수평줄을 삽입할 때 사용
 - 닫는 태그는 없음


#  인용문을 넣는 <blockquote> 태그
 - 다른 블로그나 사이트의 글을 인용할 경우, <blockquote> 태그를 이용해 표시
 - 인용한 문장은 다른 텍스트보다 안으로 들여 써지므로 다른 텍스트와 구별됨
 - 너비와 높이 지정 가능
 - 줄 개행 o


# 입력하는 그대로 화면에 표시하는 <pre> 태그
 - <pre> 태그를 사용할 경우, 소스에 표시한 공백이 브라우저에 그대로 표시됨
 - 태그 까지도 그대로 표시
 - 줄 개행이 되고 문자도 인식
 - <code> 태그도 비슷한 역할. 입력하는 그대로 화면에 표시하기. HTML 태그 포함


# 텍스트를 한 줄로 표시하는 태그
 - inline레벨 태그
 - 인라인 레벨 태그
 - 줄바꿈 없이 텍스트를 표시
 - 스타일에서 너비와 높이를 지정 불가
 - 상하여백(높이)은 지정하지 못하지만 좌우여백은 설정 가능
 - 문장이라고 생각
 - 문장은 사이의 띄어쓰기만 가능하니까라고 생각하면 됨


# 굵게 표시할 때 사용하는 <strong> 태그, <b> 태그
 - <strong> 굵게 강조할 텍스트 </strong>
 - <b>굵게 표시할 텍스트</b>


# 이탤릭체로 표시할 때 사용하는 <em>태그, <i>태그 
 - <i>이탤릭체로 표시할 텍스트</i>
 - <em>이탤릭체로 강조할 텍스트</em>


# 인용 내용을 표시하는 <q> 태그
 - 인용할 때 사용하는 태그
 - 인라인 레벨 태그
 - 줄 개행 X
 - <q>태그 가 <blockquote>태그와 다른 점은 <blockquote> 태그는 블록 레벨 태그이기 때문에 인용 내용이 줄이 바뀌어 나타나고 다른 내용과 구별되도록 안으로 들여 써짐
 - 반면 <q> 태그는 인라인 레벨 태그이기 때문에 줄바꿈 없이 다른 내용과 함께 한 줄로 표시되고, 인용 내용에 따옴표를 붙여 표시한다


# 그 밖의 태그들
 - 형광펜 표과를 내는 <mark> 태그
 - 줄바꿈 없이 영역을 묶는 <span> 태그
 - 취소선을 나타내는 <S>, <strike>, <del> 태그
 - 위첨자를 나타내는 <sup> 태그
 - 아래첨자를 나타내는 <sub> 태그


* 참고
 - div : 블록 레벨 태그
 - span : 인라인 레벨 태그 


# 목록을 만드는 태그
 - 순서 없는 목록을 만드는 <ul> 태그, <li> 태그 
 - 순서 있는 목록을 만드는 <ol> 태그, <li> 태그


# list-style-type
 - disc : 기본값. 채워진 동그라미
 - circle : 채워지지 않은 원
 - square : 채워진 네모
 - none : 아무것도 표시하지 않음. 기본 들여쓰기 되어 있음 


# <ol> 태그 속성으로 순서 목록의 숫자와 순서 바꾸기
 - type 속성으로 숫자의 종류를 다양하게 조절할 수 있음
 -  1	숫자(기본값)
 -  a	영문 소문자
 -  A	영문 대문자
 -  i	로마숫자 소문자
 -  I	로마숫자 대문자
 - start 속성 : 순서 목록은 기본적으로 1부터 시작하지만, start 속성을 이용하면 중간 번호부터 시작할 수 있음
 - reversed 속성 : 항목을 역순으로 표시


# 설명 목록을 만드는 <dl>, <dt>, <dd> 태그
 - <dl> 목록을 만드는 <dl> 태그
 - <dt> 제목을 만드는 <dt> 태그
 - <dd> 설명을 표시하는 <dd> 태그
 - 하나의 <dt> 태그에 여러 개의 <dd> 태그 값을 가질 수도 있고, 여러 개의 <dt> 태그를 가질 수도 있음


# 기본적인 표 만들기
 - <table>, <tr>, <td>, <th> -> 기본적인 표 만들기 태그
 - 표 = 행과 열로 구성


# 표를 만드는 태그
 - 기본적인 표를 만드는 <table>, <tr>, <td>, <th> 태그


# 테이블 만드는 방법
 - 1. <table> 태그로 표 자리를 먼저 만들기
 - 2. <tr> 태그로 행을 만들기
 - 3. <td> 태그로 각 행마다 셀을 만들기
 - 4. 각 셀에 들어갈 내용은 <td>와 </td> 사이에 입력

    <table>
        <tr>
            <td>...</td>
            <td>...</td>
            <td>...</td>
        </tr>
        <tr>
            <td>...</td>
            <td>...</td>
            <td>...</td>
    </tr>
    </table>


# 표에 제목 셀을 만드는 <th> 태그
  - <td> 태그와 마찬가지로 셀을 만드는 태그로, 해당 셀에 들어가는 내용을 셀의 중앙에 배치하고 굵게 표시 (굵게 중앙 가운데 정렬)
  - 각각의 제목을 의미

    <table>
        <tr>
            <th>제목 셀</th>
            <td>1행 2열</td>
            <td>1행 3열</td>
        </tr>
        <tr>
            <th>제목 셀</th>
            <td>2행 2열</td>
            <td>2행 2열</td>
        </tr>
    </table>


# 표 구조를 정의할 수 있는 <thead>, <tbody>, <tfoot> 태그
  - table의 t와 제목 부분(head), 본문(body), 요약 부분(foot)이란 말이 합쳐진 <thead>와 <tbody>, <tfood> 태그
    - <thead> : 표의 제목 영역
    - <tbody> : 표의 데이터 영역
    - <tfoot> : 표의 요약 정보

        <thead>
            <tr> ... </tr>
        </thead>
        <tbody>
            <tr> ... </tr>
        </tbody>
        <tfoot>
            <tr> ... </tr>
        </tfoot>


# 행과 열을 합치는 colspan, rowspan 속성 
 - 행과열을 합치는 colspan, rowspan 속성
    * colspan : 열 합치기 
        (여러 열을 하나로 합치려면 <td> 태그나 <th> 태그 안에서 colspan 속성을 사용해 몇 개의 셀을 가로로 합칠지 지정)
    * rowspan : 행 합치기
        (<td>태그나 <th> 태그에서 rowspan 속성을 이용해 새로 합칠 셀의 개수를 지정)


# 표에 제목을 붙이는 <caption> 태그, <figcaption> 태그      
 - 표에 제목을 붙이는 <caption> 태그, <figcaption> 태그
 - <caption>표 제목</caption> 
        caption : <table> 태그 바로 다음에 사용. <table>바로 안쪽 아래, caption 태그를 사용한 표 제목은 표의 바로 위쪽 중앙에 표시
 - <figcaption>표 제목</figcaption> 
        figcaption : figure와 caption의 합성어. <table> 태그 위에 사용. figcaption 태그를 사용한  표 제목은 표 위에 왼쪽정렬 되어 표시


# 여러 열 묶어 스타일을 지정하는 <col>, <colgroup> 태그
  - <col>, <colgroup> 태그
    * 표에서 하나의 열에 스타일을 지정하거나 열(column)을 몇 개씩 묶어 스타일을 한꺼번에 지정할 수도 있음
    * <col> 태그를 사용해 둘 이상의 열을 묶어 같은 스타일을 지정하려면, span 속성을 이용해 몇 개를 함께 묶어줄지 지정할 수 있음
    * span : span"2" -> 2개 열에 동시에 적용


# th, td 내용 정렬
 - align : left 좌측정렬, right 우측 정렬, center 가운데 정렬
 - valign : top 위쪽 정렬, bottom 아래쪽 정렬, middle 가운데 정렬
